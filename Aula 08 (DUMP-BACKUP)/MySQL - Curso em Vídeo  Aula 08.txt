Steps to make a Dump (backup of a database archive of MySQL Data Base) on MySQL:

	1. On the menu, acess "Server";
	2. Click on "Data Export";
	3. Select schemas/database and his objects (tables etc);
	4. Select the type of Dump (Structure and Data, Data only or 
	5. Dump structure of table only);
	6. In export options, you gonna select export as folder or only one archive;
	7. Mark the box "Include Create Schema" to save the process of create database 
	by himself when the Dump is done;
	8. Start export;
	9. Checkout the folder that has been saved the SQL File and open with
	a text editor or something. Inside the archive, the script genereted as
	Dump, will show all steps executed to created that database.

So, to import a Dump saved, just click on "Server", search for Data Import, click on "Import from Self-contained file" and select path/archive you want to import. And then, refresh schemas.
